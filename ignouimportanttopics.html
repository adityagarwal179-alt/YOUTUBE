Question 1: Flowcharts & Algorithms 5 Marks

Prompt: Draw a flowchart and write an algorithm to print the factorial of a number entered by the user.

Algorithm Solution:

  1. Start
  2. Read the target integer variable n from the user
  3. Initialize tracking accumulator structures: i = 1, fact = 1
  4. Repeat Steps 5 and 6 while loop index condition i <= n remains True
  5. Calculate: fact = fact * i
  6. Increment counter: i = i + 1
  7. Print/Display the final evaluated value inside fact
  8. Stop

Flowchart Layout Guide:

  • Use an Oval (Capsule) container to mark your Start and Stop nodes.
  • Place your read statement (Input n) inside a slanted Parallelogram container.
  • Insert structural preparation definitions (i=1, fact=1) inside a standard Rectangle container.
  • Use a horizontal Diamond Box to check your loop boundary condition (Is i <= n?). The Yes branch updates the calculations and loops back; the No branch prints the output and halts.

Question 2: Advanced Programming Paradigms 6 Marks

Prompt: What are lambda functions? Discuss their utility. Write a lambda function in Python to calculate the cube of a number entered by the user.

Answer:

  • Definition: A lambda function is an anonymous, unnamed function in Python that is defined inline using the lambda keyword. It can take any number of arguments but can only contain a single expression.
  • Utility: Lambda functions are used for short, throwaway blocks of logic where declaring a full function with def is unnecessary. They are frequently used inside higher-order functions like map() and filter().

Python Implementation:

# Prompting user input
user_num = float(input("Enter a number to cube: "))

# Inline Lambda calculation sequence
cube_operation = lambda x: x ** 3

# Execution and presentation 
result = cube_operation(user_num)
print(f"The cube of {user_num} is: {result}")

Question 3: Structural Parameter Passing 6 Marks

Prompt: Explain the concept of Call by Value with suitable code in C. Give its advantages and disadvantages.

Answer:

In a "Call by Value" scheme, the actual values of the arguments are copied into the formal parameters of the called function. This creates entirely distinct, isolated memory locations. Modifications made inside the function have no effect on the original variables.

C Sample Implementation:

#include <stdio.h>

// Function declaration
void modifyValue(int num) {
    num = num + 100; // Modifies only the local copy
    printf("Inside function frame, local variable = %d\n", num);
}

int main() {
    int original = 50;
    printf("Before calling function, original variable = %d\n", original);
    
    modifyValue(original); // Passing by value
    
    printf("After calling function, original variable = %d\n", original);
    return 0;
}
  • Advantages: It is highly secure, prevents accidental data corruption, and keeps variables completely isolated.
  • Disadvantages: Copying large items (like structures or large datasets) consumes extra execution time and memory overhead.

Question 4: Object-Oriented Implementations 5 Marks

Prompt: What is method overloading and method overriding in Python? Provide clear code examples for each.

Answer:

Method Overloading: Simulated in Python using default parameter assignments (like None) within a single method block, as Python does not support classic compile-time method overloading directly.

class Calculator:
    def compute_sum(self, a, b, c = None):
        if c is not None:
            return a + b + c
        return a + b

obj = Calculator()
print(obj.compute_sum(10, 20))      # Outputs 30
print(obj.compute_sum(10, 20, 30))  # Outputs 60

Method Overriding: Occurs when a child class redefines a method that already exists in its parent class to provide a more specific action profile.

class Parent:
    def display_message(self):
        print("This is the Parent Class execution.")

class Child(Parent):
    def display_message(self):
        print("This is the Child Class overridden execution.")

child_instance = Child()
child_instance.display_message() # Runs Child implementation

  1. Focus on Program Structure: When writing C code on your answer script, make sure to include preprocessor lines (#include <stdio.h>), define variable types clearly, and format your output using printf and scanf. For Python, maintain clean layout indentations.
  2. Practice Iteration Code Duals: Ensure you can confidently convert iterative loops into equivalent recursive logic. Practicing standard loops (like computing sums up to n) using both methods is crucial.
